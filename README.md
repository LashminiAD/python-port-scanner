# Port Scanner (Python)

## Description
A Python-based port scanner that detects open ports on a target system using TCP connections.

## Features
- Scans ports from 1–200
- Detects open ports
- Handles errors (invalid host, network issues)
- Displays results clearly
- Beginner-friendly implementation

## How it Works
The scanner:
1. Takes a target IP/domain
2. Attempts TCP connection on each port
3. Identifies open ports based on response

## Technologies Used
- Python
- Socket Programming

## How to Run
```bash
python port_scanner.py
```
